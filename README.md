# Client Server Chat Application

A simple terminal-based chat application built using Python that demonstrates the Client-Server networking model.

## Project Overview

This project allows multiple clients to connect to a central server and communicate with each other in real time using terminal windows.

## Technologies Used

- Python
- Socket Programming
- Threading

## How to Run the Project

### Step 1: Start the Server

python server.py

### Step 2: Start the Client

Open another terminal and run:

python client.py

Enter your name and start chatting.

## Features

- Real-time terminal chat
- Multiple client support
- Client–Server architecture
- Lightweight and simple implementation
